Linux PETSc/MPI tutorial - VoxFE-UV 0.31
========================================

This release is Linux-only and adds an experimental direct PETSc/MPI assembly path.

What changed
------------

Version 0.30 assembled a SciPy sparse matrix first and then converted it to PETSc.
Version 0.31, when launched with --backend petsc_optional, assembles a distributed
PETSc AIJ matrix directly. Each MPI rank owns a row range and inserts only the
element stiffness rows belonging to that rank.

Recommended environment
-----------------------

Use Linux or WSL2. The easiest installation path is conda-forge:

   mamba create -n voxfe-petsc-mpi -c conda-forge python=3.12 numpy scipy petsc petsc4py mpi4py numba pyamg
   conda activate voxfe-petsc-mpi

Optional viewer dependencies:

   conda install -c conda-forge pyvista vtk pillow imageio

Verify PETSc/MPI:

   python -c "from petsc4py import PETSc; print(PETSc.Sys.getVersion(), PETSc.COMM_WORLD.getSize())"
   mpiexec -n 2 python -c "from petsc4py import PETSc; print('rank', PETSc.COMM_WORLD.getRank(), 'of', PETSc.COMM_WORLD.getSize())"

Install VoxFE-UV as an Ubuntu app
---------------------------------

From the release root:

   sh install_ubuntu_app_0_31.sh

This installs a per-user application launcher:

   ~/.local/share/applications/voxfe-uv.desktop

and a terminal command:

   voxfe-uv

The release folder remains the actual application folder. Do not delete or move it
after installing the launcher unless you reinstall the launcher from the new path.

To remove the app-menu entry and terminal command:

   sh uninstall_ubuntu_app_0_31.sh

Run one model
-------------

Go inside one model folder:

   cd models/P1_linear_25

Run with the default PETSc GAMG method on 2 MPI ranks:

   sh ../../solve_this_folder_petsc_mpi_0_31.sh

Use more ranks:

   MPI_PROCS=4 sh ../../solve_this_folder_petsc_mpi_0_31.sh

Try Jacobi-preconditioned PETSc CG:

   METHOD=PETSCJAPCG MPI_PROCS=4 sh ../../solve_this_folder_petsc_mpi_0_31.sh

Try direct LU only on small models:

   METHOD=PETSCLU MPI_PROCS=2 sh ../../solve_this_folder_petsc_mpi_0_31.sh

Compare PETSc/MPI methods
-------------------------

Inside a model folder:

   sh ../../compare_petsc_mpi_this_folder_0_31.sh

Custom comparison:

   METHODS=PETSCGAMG,PETSCJAPCG MPI_PROCS=4 sh ../../compare_petsc_mpi_this_folder_0_31.sh

Outputs
-------

The solver writes from MPI rank 0 only:

   displacement.txt
   displacement_standard.txt
   displacement_standard_with_coords.txt
   force.txt
   sed.csv
   residual_history.csv
   voxfe_solver_summary_petsc_mpi_60000.json
   solver_petsc_mpi_60000_stdout.log
   coordinate_export_summary_petsc_mpi.json

Important notes
---------------

This is an experimental MPI path. The model and boundary-condition parsing are still
replicated on each rank, but the stiffness matrix is assembled directly as a PETSc
distributed AIJ matrix and solved with PETSc KSP/PC.

For large models, start with PETSCGAMG and 2-4 MPI ranks. Increase ranks only after
checking that memory use and residual history improve on your machine.
